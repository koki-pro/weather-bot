# LINEで雨が降るかどうか教えれくれるbot

このbotは、毎朝７時に東京で雨が降るかどうかを通知してくれるbotです</br>
降水確率が20%以上であれば、傘を持っていくように通達してくれます

## 本アプリの環境
・ruby 2.6.5</br>
・rails 6.0.3</br>
・postgresql
